# Talk to Your Infra: Overview

The **Chat interface**  is RubixKube's conversational AI that lets you interact with your infrastructure using natural language. No more memorizing kubectl commands or navigating complex dashboards - just ask questions like you would to a colleague.

<Info>
**Complete chat mastery in 3 tutorials:** We've split this into focused guides so you can learn at your own pace or jump to what you need.
</Info>

## Quick Example

**Traditional way:** ```bash
kubectl get pods -n production
kubectl describe pod api-gateway-xyz
kubectl logs api-gateway-xyz
kubectl top pod api-gateway-xyz
# ... 5 more commands ...
```

**Chat way:** ```
"Why is api-gateway slow?"
```

**Result:** Complete RCA in 30 seconds instead of 10+ minutes.

## Quick Reference

**Most common queries:** 1. `"What's failing?"` - Daily triage
2. `"Why is [pod] crashing?"` - Troubleshooting
3. `"Cluster health?"` - Health check
4. `"Validate this"` - Pre-deploy (with YAML upload)
5. `"Explain [concept]"` - Learning

**Fastest way to start:** Press `⌘K`, type "chat", ask away!
