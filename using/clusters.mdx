## Connecting a Cluster

<Steps>
  <Step title="Click Connect Cluster">
    Opens installation dialog.
  </Step>

  <Step title="Copy kubectl Command">
    Command includes your API key. Keep it secret.
  </Step>

  <Step title="Run in Your Cluster">
    ```bash
    kubectl apply -f "https://api.rubixkube.ai/install/observer.yaml?apiKey=YOUR_KEY"
    ```
  </Step>

  <Step title="Verify Connection">
    Cluster appears in list within 1-2 minutes
  </Step>
</Steps>

## Cluster Status

| Status | Meaning |
|--------|---------|
| Connected | Healthy, actively monitored |
| Degraded | Partial connectivity |
| Disconnected | No connection |
