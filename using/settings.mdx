## Settings Tabs

### Organization

**What you'll find:**
- **Workspace name:** "Connect's Group"
- **Plan tier:** free, pro, enterprise
- **Active users:** Team member count
- **Clusters:** Connected cluster count
- **Contact email:** Organization email
- **Tenant ID:** Your unique RubixKube identifier

**Use case:** View subscription and organization overview.

---

### Billing & Usage

**What you'll find:**
- Current plan details
- Usage statistics
- Upgrade options

**Use case:** Monitor usage and manage subscription.

---

### Preferences

**What you'll find:**
- **Display name:** Your name
- **Email:** Account email  
- **Role:** admin, member, viewer
- **User ID:** Your unique identifier
- **Email status:** Verified/Unverified

**Use case:** Update personal information.

---

### System

**What you'll find:**
- RubixKube version
- Agent versions
- API endpoint status
- System health checks

**Use case:** Verify system status, check versions.

---

### Security

**What you'll find:**
- **Authentication method:** SSO (Auth0)
- **Password status:** OAuth or password-based
- **Security actions:** Enable/disable features
- **Audit logs:** Recent security events

**Use case:** Manage authentication and security.

---

### Integrations

**What you'll find:**
- Connected services (Slack, PagerDuty, etc.)
- API keys for integrations
- Webhook configurations

**Use case:** Manage third-party connections.

---

## Common Tasks

### Copy Tenant ID

1. Go to Organization tab
2. Find "Tenant ID"  
3. Click copy button
4. Use for API calls or support requests

### Update Profile

1. Go to Preferences tab
2. Edit Display Name
3. Changes save automatically

### Check Authentication

1. Go to Security tab
2. Verify authentication method
3. See if using OAuth (recommended) or password
